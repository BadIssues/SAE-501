JALON 17 : EDGEX NAT/PAT VPN
============================

IMAGE : J17-EDGE1-nat-config.png
DESCRIPTION : Configuration NAT statique sur EDGE1.
POURQUOI : Montre les règles de redirection de port pour le VPN et le Webmail.

IMAGE : J17-EDGE1-loopback.png
DESCRIPTION : Interface Loopback0 (IP Provider Independent).
POURQUOI : Cette IP publique est utilisée comme destination pour les services hébergés.

IMAGE : J17-EDGE1-routes-null0.png
DESCRIPTION : Route statique vers Null0 pour l'IP Loopback.
POURQUOI : Nécessaire pour que le BGP annonce correctement le réseau /28.

IMAGE : J17-EDGE2-nat-config.png
DESCRIPTION : Configuration NAT statique sur EDGE2.
POURQUOI : Assure la redondance des services si EDGE1 tombe.

IMAGE : J17-EDGE2-loopback.png
DESCRIPTION : Interface Loopback sur EDGE2.
POURQUOI : Vérification de la configuration identique pour la haute disponibilité.

IMAGE : J17-internet-ping-loopback.png [MANQUANT]
DESCRIPTION : Ping depuis Internet vers l'IP publique de service.
POURQUOI : Prouve que les services sont joignables depuis l'extérieur.
